## Mistake Format

When adding a new mistake file, use this format:

```markdown
# Mistake: Name

## What Happened
Brief description of the mistake.

## Why It Was Wrong
Explanation of the problem.

## Better Approach
What should have been done instead.

## Example

Bad:
```code
Code that caused the problem
```

Good:
```code
Code that would have worked
```

## How to Detect
Signs that you might be making this mistake.

## Prevention
How to avoid this in the future.
```

## Adding to Index

Add to this file in format:
```
mistake-name.md -- Short description -- tag1, tag2, tag3
```
